## v0.2.1 (2026-06-10)

### 🏗 模块化重构

- **多文件拆分** — 将原来 2376 行的 `disk_cleaner.py` 拆分为 6 个独立模块：
  - `config.py` — 全局常量、权限检测、Windows 特殊文件夹
  - `utils.py` — 工具函数（格式化、文件夹大小、回收站操作）
  - `clean_tasks.py` — 六大清理任务类
  - `tray.py` — 系统托盘图标（纯 ctypes）
  - `log_manager.py` — 日志管理器（单例、每日轮转）
  - `disk_cleaner.py` — 精简为主入口，仅保留 GUI 界面 + 启动函数
- 更新 `DiskCleaner.spec` 和 `build.bat` 以包含所有新模块
- 更新所有文档（`README.md`、`ARCHITECTURE.md`、`CHANGELOG.md`）反映新结构

### 🐛 路径修复

- **`WINDIR` 默认值** — 去掉硬编码的 `"C:\\Windows"`，改用 `os.environ` 读取
- **`Downloads` 路径** — 增加中文系统「下载」目录兼容
- **`_documents()` 方法** — 增加 `SHGetFolderPathW(CSIDL_PERSONAL)` API 获取真正的用户文档目录，提升跨语言系统兼容性

### 🚀 重大功能更新

#### 📱 应用软件缓存清理（全新模块）
- **7 款主流软件支持** — 自动检测微信、QQ、TIM、企业微信、钉钉、飞书、百度网盘、WPS、迅雷的缓存目录
- **文件级选择粒度** — 每个软件可展开查看细分的缓存类别（图片、视频、语音、日志等），逐项独立勾选
- **`app_cleaner.py`** — 全新独立模块，包含完整的路径发现、扫描、清理逻辑
- **智能发现** — 自动查找微信 `wxid_*` 目录、QQ 号目录、钉钉/飞书/百度网盘等多安装路径
- **展开/折叠 UI** — 软件分类可折叠，界面清爽不占空间
- **分类扫描** — 每个软件旁有独立「扫描」按钮，也可一键「扫描全部」
- **配置持久化** — 每个缓存项的勾选状态独立保存到配置文件

#### 多语言国际化（i18n）
- **中英双语支持** — 程序自动检测 Windows 系统语言（中文 / English）

- 智能回退：找不到语言包时自动 fallback 到英文
- 扩展新语言只需添加 JSON 文件 + 注册字典条目

## v0.1 (2026-06-09)

### 新增功能

#### 六大清理模块
- **📁 系统临时文件** — 清理 `%TEMP%`、`%WINDIR%\Temp`、`%LOCALAPPDATA%\Temp`
- **♻ 回收站** — `SHQueryRecycleBinW` 查询大小，`SHEmptyRecycleBinW` 一键清空
- **🌐 浏览器缓存** — 兼容 Chrome / Edge 的 Cache、Code Cache、CacheStorage
- **📥 下载文件夹旧文件** — 用户可自定义天数（默认 30 天），按修改时间过滤
- **📋 最近访问记录** — SHAddToRecentDocs API + Recent 文件夹 .lnk 清理
- **📜 系统日志文件** — C:\Windows\Logs、CBS、Panther、.etl、memory.dmp 等

#### GUI 界面
- 基于 tkinter 的原生 Windows 界面
- 可滚动任务卡片列表，每项独立复选 + 扫描按钮 + 实时大小显示
- 全选 / 全不选 / 扫描全部 / 开始清理 / 模拟运行
- 暗色主题日志文本框（支持滚动、清空、导出）
- 清理进度条 + 状态文字
- 旧文件天数输入框（仅下载任务显示）

#### 安全机制
- 默认「移动到回收站」模式，可切换为永久删除
- 清理前计算总大小并弹窗确认
- 未扫描项目自动跳过提醒
- 所有文件操作均有 `try/except` 保护

#### 权限管理
- 启动时自动检测管理员权限
- 非管理员自动弹出 UAC 提权请求（`ShellExecuteW runas`）
- 用户拒绝 UAC 时降级为普通权限继续运行
- 标题栏显示当前权限状态（管理员/普通用户）
- 清理系统日志前二次检查权限

#### 系统托盘
- 纯 `ctypes` 实现，零外部依赖
- 关闭窗口最小化到托盘
- 双击托盘图标恢复窗口
- 右键菜单：显示窗口 / 退出程序
- 清理完成托盘气泡通知
- 可选启用/禁用

#### 日志系统
- 每日自动轮转文件日志：`%TEMP%\DiskCleaner\logs\DiskCleaner_YYYY-MM-DD.log`
- 四级日志：DEBUG / INFO / WARN / ERROR
- 线程安全写入（Lock 保护）
- GUI 文本框实时同步
- 自动清理 30 天前的旧日志
- 一键打开日志文件夹按钮
- 启动时记录版本、权限、日志路径等系统信息

#### 配置持久化
- 用户配置保存至 `~/.disk_cleaner_config.json`
- 持久化内容：回收站模式、托盘偏好、天数、任务选中状态
- 窗口关闭 / 托盘退出时自动保存

### 技术细节
- **语言**: Python 3.8+
- **GUI**: tkinter（标准库）
- **Win32 API**: ctypes（Shell32 / User32 / Kernel32）
- **打包**: PyInstaller `--onefile --noconsole`
- **总代码量**: ~2000 行
- **exe 体积**: ~13 MB

### 已知问题
- 浏览器缓存扫描对大量小文件时较慢（`os.walk` 限制）
- Windows 7 未完整测试
- 32 位系统需自行用 x86 PyInstaller 打包
